## When to use this skill

Use this skill when:
- You have competitor search results that need interpretation
- User asks for strategic analysis of competitor moves
- Comparing competitor positioning against the user's business
- Determining if a competitor change warrants action

## Analysis Framework

### Significance Scoring (1-10)

| Score | Level    | Description                                      |
|-------|----------|--------------------------------------------------|
| 1-3   | Low      | Routine updates, minor mentions                  |
| 4-6   | Medium   | Notable changes that may affect market            |
| 7-9   | High     | Major strategic moves, product launches, pivots   |
| 10    | Critical | Industry-disrupting events, direct threats        |

### Analysis Dimensions

1. **Market Impact**: How does this affect the competitive landscape?
2. **Relevance**: How directly does this affect the user's business?
3. **Urgency**: How quickly should the user respond?
4. **Actionability**: What concrete steps can the user take?

## Output Format

Provide analysis as:
- **Summary**: 1-2 sentence overview
- **Significance**: Score and reasoning
- **Key Findings**: Bullet points of important details
- **Recommended Actions**: What the user should consider doing
- **Watch Items**: Things to keep monitoring
